Primtive Datatype in js

String: Represents text data enclosed in single (' '), double (" "), or backticks (` `) for template literals.

number: Represents both integer and floating-point numbers

boolean: Represents a logical entity with two possible values: true or false.

undefined : Represents a variable that has been declared but has not yet been assigned a value

null: Represents the intentional absence of any object value. It’s often used to indicate “no value” or “empty.”

bigint: Used for representing integers with arbitrary precision. It allows for safe operations on large integers beyond the range of Number.

symbol: A unique and immutable data type used primarily to create unique keys for object properties.
